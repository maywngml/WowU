# WowU (Workout with U)

### Summary
 프로젝트 Logo


#### * 인공지능을 이용한 홈트레이닝 자세교정 어플리케이션
#### * Keypoint Detection을 이용해 사용자 몸을 실시간으로 인식
#### * 선택 자에 맞는 사용자 맞춤 가이드라인 제공


#### 구현된 기능
##### 1.10초마다 사용자 몸을 인식해 가이드라인을 그린다. 
##### 2.원활한 운동을 위해 **음성으로 자세별 주의사항을 설명한다. 
##### 3.스쿼트, 와이드스쿼트 자세 구현 완료
##### 4. 사용자 몸을 인식해 운동횟수 카운트

#### 추가기능
##### 1. 더 많은 홈트레이닝 자세 필요
##### 2. 운동횟수 카운트 기능 보완 필요
